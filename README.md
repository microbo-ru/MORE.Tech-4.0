## Как запустить

### Install backend
1. Run docker-compose up -d command to run database
2. Run  mvn spring-boot:run command to run backend app

### Install frontend
1. Go to WEB-APP folder
2. Run npm i -g yarn
3. Run yarn install
4. Run npm start
